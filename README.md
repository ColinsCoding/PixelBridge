# PixelBridge
A Pixel Editor built with the p5.js library. 

Features: 
Implemented; 
Brushes
  Paint Pen - sets the brush to a small pixel size and no faded edges 
  Sprary Paint - creates a large brush with faded edges 
Multiple Canvases 
  Drawing Canvas 
  A posting canvas 

Planned:
Saves as JPEG or PNG,
Selection Tool
